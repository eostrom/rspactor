=== Install

  git clone git://github.com/guillaumegentil/rspactor.git
  cd rspactor
  rake install

=== Usage

  cd /path/to/rails/root/
  rspactor

=== Command Line Options

* <code> --coral    </code>
* <code> --drb      </code>
* <code> --view     </code>
* <code> --clear    </code>
* <code> --run_in   </code>
