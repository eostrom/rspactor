=== Dependencies

RubyCocoa is needed (you can help to switch to ruby-fsevent, see branches)

  Installing ruby-cocoa with rvm: http://gist.github.com/294465

=== Install

  gem install rspactor

=== Usage

  cd /path/to/rails/root/
  rspactor

=== Command Line Options

* <code> --coral    </code>
* <code> --drb      </code>
* <code> --view     </code>
* <code> --clear    </code>
* <code> --run_in   </code>
